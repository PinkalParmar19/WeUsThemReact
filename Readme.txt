	- validator (for email validation)
	- semantic-ui (for style)
	- npm i cors (for crs policy) req. and response cross sharing..

	2. How to run app:
	a. install node.js
	b. open editor of your choice
	c. select folder phonebook from destination and start terminal
	d. type npm start in the terminal (make sure you are in the right directory- use cd)
